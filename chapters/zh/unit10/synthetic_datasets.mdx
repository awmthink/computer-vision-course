# 合成数据集

## 介绍
欢迎来到计算机视觉中合成数据集的精彩世界！随着我们从经典的无监督方法转向高级深度学习技术，对大量和多样化数据集的需求迅速增加。合成数据集已成为训练最先进模型的重要资源，提供了在现实世界中通常无法收集的丰富数据。在本节中，我们将探讨一些最具影响力的合成数据集，它们的应用，以及它们如何塑造计算机视觉的未来。

## 低层次计算机视觉问题

### 光流和运动分析

光流和运动分析对于理解图像动态至关重要。以下是一些在该领域中有显著贡献的数据集：

| 数据集名称           | 年份                  | 描述                                                  | 论文                                                        | 附加链接                                             |
| ---------------------- | --------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| Middlebury             | 2021 (最新发布)      | **Middlebury** Stereo 数据集包含复杂几何结构和像素级准确的地面真值视差数据的高分辨率立体序列。地面真值视差是通过使用结构光的技术获得的，不需要校准光投影仪。 | [A database and evaluation method for Optical Flow](https://vision.middlebury.edu/flow/floweval-ijcv2011.pdf) (撰写时引用3192次) | [Papers with Code](https://paperswithcode.com/dataset/middlebury) - [Website](https://vision.middlebury.edu/stereo/data/) |
| Playing for Benchmarks | 2017                  | 超过25万张高分辨率视频帧，所有都带有高层次任务的地面真值数据，以及低层次任务的光流估计和视觉里程计。 | [Playing for benchmarks](https://arxiv.org/abs/1709.07322)   | [Website](https://playing-for-benchmarks.org/)               |
| MPI-Sintel             | 2012                  | 用于光流的合成数据集。**MPI-Sintel** 的主要特点是包含相同场景但不同渲染设置的图像，这种方法有助于深入了解不同光流算法在何处失效。(论文引用) | [A Naturalistic Open Source Movie for Optical Flow Evaluation](https://link.springer.com/chapter/10.1007/978-3-642-33783-3_44) (撰写时引用551次) | [Website](http://sintel.is.tue.mpg.de/)                      |
